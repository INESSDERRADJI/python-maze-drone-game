# Python Maze Drone Game
> This repository presents a portfolio overview of the project. The source code is not publicly available.

Python game project featuring a maze, moving drones, obstacles and target zones.

## Overview

This project is a maze-based game where the player controls a drone and must reach target areas while avoiding obstacles and enemy drones.

The game includes two types of enemies: simple drones with basic movement patterns, and an intelligent enemy drone that follows the player even when the player changes direction.

## Main Features

- Maze-based gameplay
- Player-controlled drone
- Obstacles and target zones
- Simple enemy drones with basic movement
- Intelligent enemy drone with path-following behavior
- Score and health system
- Warning message when an enemy is close

## Algorithmic Focus

The most complex part of the project was the intelligent enemy behavior.  
Unlike the basic drones, the smart enemy dynamically follows the player through the maze and adapts its path when the player changes direction.
