# Python Maze Drone Game

> This repository contains a simplified project skeleton. The full final source code is not publicly available.
Python maze game prototype with a player-controlled drone, obstacles, target zones and enemy drones.

## Overview

This repository contains a simplified code skeleton of an academic Python game project.  
The original project included basic enemy drones and a smarter enemy behavior designed to follow the player through the maze.

## Features

- Maze-based gameplay structure
- Player-controlled drone
- Obstacles and target zones
- Enemy drone behavior concept
- Score and health system
